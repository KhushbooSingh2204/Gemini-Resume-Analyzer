# AI Resume Analyzer using Gemini

## Overview

AI Resume Analyzer is a Streamlit-based web application that uses Google's Gemini AI to analyze resumes and provide detailed feedback. Users can upload a resume in PDF format, specify a target job role, and receive AI-generated recommendations to improve their resume.

## Features

* Upload Resume PDF
* Extract Resume Content
* Analyze Resume for a Specific Job Role
* Identify Key Strengths
* Detect Missing Skills
* Highlight Weak Areas
* Provide Improvement Suggestions
* Assess Interview Readiness
* Generate Overall Resume Evaluation

## Technologies Used

* Python
* Streamlit
* Google Gemini AI
* PyPDF

## How It Works

1. Enter your Gemini API Key.
2. Upload your resume in PDF format.
3. Enter the target job role.
4. Click the "Analyze Resume" button.
5. Gemini AI analyzes the resume and generates detailed feedback.

## Installation

Install the required libraries:

```bash
pip install streamlit
pip install pypdf
pip install google-generativeai
```

## Run the Application

```bash
streamlit run resume_analyzer.py
```
